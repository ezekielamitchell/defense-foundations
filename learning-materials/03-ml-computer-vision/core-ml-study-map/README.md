# Core ML Study Map

**Activation:** P2

## Sequence

Linear/logistic regression → trees and ensembles → k-NN/SVM → clustering/PCA → cross-validation → ROC/PR → calibration → threshold selection → leakage and failure analysis.

## Tools

scikit-learn, pandas, seaborn, and script-first experiments. Notebooks may be used for exploration, but final evidence must be reproducible outside a notebook.

## Evidence

A calibrated, low-false-positive tabular classifier with multiple operating points and a defense-relevant FP/FN cost discussion.
