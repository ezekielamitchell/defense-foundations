# Pro Git

**Source:** [official online book](https://git-scm.com/book/en/v2)  
**Cost:** free  
**Activation:** P0, W28

## Chapters

1. Getting Started.
2. Git Basics.
3. Git Branching.

## Evidence

Create and merge a short-lived branch, inspect the diff before committing, and document the commands used. The goal is fluent status/diff/branch/commit/push behavior, not memorizing Git internals.
