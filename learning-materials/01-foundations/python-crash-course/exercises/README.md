# Exercises — Python Crash Course, Part I

Scratch space for Chapters 1–8. One folder per chapter; add files as you work
through each chapter's "Try It Yourself" sections.

Run any file directly:

    python3 exercises/ch01_getting_started/hello_world.py

| Ch | Topic | Folder |
|---:|---|---|
| 1 | Getting started | `ch01_getting_started/` |
| 2 | Variables and simple data types | `ch02_variables_and_data_types/` |
| 3 | Introducing lists | `ch03_lists/` |
| 4 | Working with lists | `ch04_working_with_lists/` |
| 5 | `if` statements | `ch05_if_statements/` |
| 6 | Dictionaries | `ch06_dictionaries/` |
| 7 | User input and `while` loops | `ch07_input_and_while_loops/` |
| 8 | Functions | `ch08_functions/` |

When the chapters are worked, build the [`evidence/`](../evidence/README.md)
script — chapter exercises alone are not completion.

Granular progress is tracked outside this repo; keep these folders to code only.
