# MIT 6.0001 — Introduction to Computer Science and Programming in Python

**Source:** [MIT OpenCourseWare](https://ocw.mit.edu/courses/6-0001-introduction-to-computer-science-and-programming-in-python-fall-2016/)  
**Cost:** free  
**Activation:** P0, W26

## Scope

Use lectures 1–4 for computational thinking, Python syntax, branching, iteration, strings, input, and small algorithmic problems. Use the accompanying questions and assignments selectively to test recall rather than to maximize completion percentage.

## Evidence

Explain the algorithm before writing it, then produce a small file-processing program without copying lecture code.
