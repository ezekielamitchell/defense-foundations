# Rustlings

**Source:** [rust-lang/rustlings](https://github.com/rust-lang/rustlings)  
**Cost:** free  
**Activation:** P0

## Scope

Complete the first 50 exercises, using the current Rustlings ordering rather than freezing this plan to version-specific exercise names. W27 targets batches 1–3; W30 targets batches 4–5.

## Method

Read the compiler message, predict the fix, make the smallest change, and note recurring ownership/error patterns. Avoid looking at solutions until a deliberate debugging attempt has been documented.

## Evidence

Completion count plus short notes on ownership, borrowing, pattern matching, `Option`, and `Result` mistakes.
