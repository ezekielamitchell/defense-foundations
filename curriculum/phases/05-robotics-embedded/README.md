# P5 — Robotics and Embedded Systems

**Status:** not started  
**Estimated duration:** 10–12 weeks at 10–15 hours/week

## Focus

Bridge laptop ML to real sensors, microcontrollers, actuators, telemetry, and safe control loops.

## Topics

Kinematics, PID, state-space intuition, I²C/SPI/UART/CAN, real-time concepts, electronics safety, Embedded HAL, Embassy, ESP/RP ecosystems, serial control, rotations, quaternions, and ODE intuition.

## Materials

See [Robotics and embedded](../../../learning-materials/05-robotics-embedded/README.md).

## Evidence target

A robot vision system, an embedded Rust sensor node, and a Rust MQTT telemetry path.

## Exit gate

Wire and read a sensor, integrate it into a logged control loop, run Rust firmware on an MCU, and make one robot drive.
