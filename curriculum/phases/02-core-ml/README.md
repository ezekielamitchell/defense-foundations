# P2 — Core Machine Learning

**Status:** not started  
**Estimated duration:** 8–10 weeks at 10–15 hours/week

## Focus

Classical ML before deep learning, with honest evaluation and explicit false-positive/false-negative costs.

## Topics

Regression, trees, boosting, k-NN, SVM, clustering, PCA, cross-validation, ROC/PR curves, calibration, leakage, bias/variance, and threshold selection. The Python stack is scikit-learn, pandas, seaborn, and script-first experiment discipline.

## Evidence target

A low-false-positive tabular threat classifier with calibrated probabilities, operating-point analysis, and a written account of leakage and failure modes.

## Exit gate

Fit, evaluate, and debug a model end to end; explain why accuracy fails on imbalanced data; select a threshold from operational cost rather than aesthetics.
