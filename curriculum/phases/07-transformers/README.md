# P7 — Transformers and Foundation Models

**Status:** not started  
**Estimated duration:** 10–12 weeks at 10–15 hours/week

## Focus

Understand attention from first principles and use modern vision foundation models without treating them as magic.

## Topics

Self-attention, ViT, DETR, MAE, DINOv2, CLIP, SAM, open-vocabulary detection, multimodal fusion, positional encoding, and self-supervised objectives.

## Materials

See [Transformers](../../../learning-materials/07-transformers/README.md) and [foundation-model research](../../../research/foundation-models-vision/README.md).

## Evidence target

Self-attention and a tiny ViT implemented from first principles, followed by a reproducible DINOv2 fine-tuning experiment and report.

## Exit gate

Explain transformer code on a whiteboard and compare ViT/CNN edge tradeoffs quantitatively.
